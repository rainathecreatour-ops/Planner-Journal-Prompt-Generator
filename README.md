# Planner & Journal Prompt Generator

A Next.js + Tailwind web app that lets users customize:
- format (daily/weekly/monthly/journal types)
- occasion
- theme
- fonts
- background patterns
- colors
- topics
- page count & size

Then it generates:
- a copy/paste AI prompt (Prompt Only mode)
- optionally, generated output via /api/generate (Generate Text mode)

## Run locally
```bash
npm install
npm run dev
